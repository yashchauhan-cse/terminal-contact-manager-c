# Terminal Contact Manager (C)

A menu-driven, terminal-based Contact Management System written in **C**.
This project demonstrates core C programming concepts such as structures, arrays, string handling, and modular logic.

---

## Features

- Add new contacts
- Edit existing contacts (name or phone number)
- Delete contacts  
- Handles duplicate names safely using index-based selection
- Search contacts by name
- View all saved contacts
- Display total number of contacts
- Input validation to prevent invalid operations
- Fixed-size storage (no dynamic memory allocation)

---

## Tech Stack

- Language: C
- Compiler: GCC (or any standard C compiler)
- Platform: Terminal / Command Line

---

## How to Run

gcc main.c -o contact_manager
./contact_manager
